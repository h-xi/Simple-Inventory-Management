## Inventory Management System

Download MySQL Workbench <br>


**To run frontend:** <br>

`cd front_end` <br>
`npm install` <br>
`npm start` <br>



**To run backend:** <br>

`cd server` <br>
`npm install` <br>
`cd databaseData` <br>
*connect to MySQL workbench* <br>
`node populateScript --create`   <br>
`cd ..` <br>
`cd utils` <br>
`node api.js` <br>
